Fix Dark Theme WPS Office Linux

Posted on Feb 9, 2021

Intro

Dari pengguna windows yang hijrah di linux, pasti mencari alternatif microsoft office, tetapi terkendala masalah format dokumen entah acak acakan atau font tidak muncul.

Maka dari itu ada alternatif yaitu, WPS Office. Selama pemakaian untuk kepentingan kuliah, WPS Office sangat membantu bagi gw, interface yang mirip dengan Microsoft Office dan merupakan salah satu program office yang sangat user firendly. WPS Office sendiri menyediakan :

  • WPS Writer
  • WPS Presentation
  • WPS Spreadsheet
  • WPS PDF

Penjelasan

Yang menurut gw sudah cukup lengkap untuk kebutuhan office.

Tetapi ada masalah yang menurut gw cukup serius, yaitu masalah jika kita mengaktifkan dark theme gtk, karena seraca default, WPS Office menggunakan Qt.

Seperti gambar di atas, kita ndak bisa liat karakter, karena WPS secara default membaca tema Qt bukan GTK.

Sebelum ke cara fixnya, perlu di ketahui mana aja yang akan di edit yaitu :

/usr/bin/wps 	(wps writer)
/usr/bin/wpp 	(wps presentation)
/usr/bin/et  	(wps spreadsheet)
/usr/bin/wpspdf (wps pdf)

Editing

Disini gw pake nano karena belom bisa pake vim :â€˜D

sudo nano /usr/bin/wps

Ganti baris gOpt= dengan script di bawah

gOpt="-style=gtk+"
export GTK2_RC_FILES=/usr/share/themes/Matcha-azul/gtk-2.0/gtkrc

Untuk Matcha-azul bisa kalian ganti dengan light theme yang akan di pakai, ingat pakai light theme ya jangan dark theme.

Lakukan hal di atas ke semua aplikasi WPS di atas.

Seperti gambar di bawah ini

Untuk wpspdf ada sedikit tambahan di bagian fuction run() yang awalnya

function run()
{

        if [ -e "${gInstallPath}/office6/${gApp}" ] ; then
                { ${gInstallPath}/office6/${gApp} "$@"; } >/dev/null 2>&1
        else
                echo "${gApp} does not exist!"
        fi
}

Menjadi

function run()
{

        if [ -e "${gInstallPath}/office6/${gApp}" ] ; then
                { ${gInstallPath}/office6/${gApp} ${gOpt} "$@"; } >/dev/null 2>&1
        else
                echo "${gApp} does not exist!"
        fi
}

Tambahan ${gOpt} di sebelah ${gApp}

Kalau sudah bisa di run WPS Officenya.

Hasil sebelum :

Hasil sesudah (fixed), di sini saya menggunakan dark theme, sedangkan wps officenya menggunakan light theme.

Sekian, semoga bermanfaat :D
