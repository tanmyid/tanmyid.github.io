Bagian ini *==> Unable to install module vboxhost/6.0.10_OSE for kernel : Missing kernel headers. adalah penyebab virtualbox tidak bisa jalan. Untuk mengatasinya lakukan instalasi kernel headers :

sudo pacman -S linux-headers

Untuk yang menggunakan kernel LTS

sudo pacman -S linux-lts-headers
