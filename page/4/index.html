Instalasi Cmder

Hay guys kali ini ada sedikit threads soal instalasi CMDER, apasi CMDER itu? CMDER adalah paket perangkat lunak yang dibuat karena frustrasi murni karena tidak adanya emulator konsol yang bagus di Windows. Ini didasarkan pada perangkat lunak luar biasa, dan dibumbui dengan skema warna Monokai dan tata letak prompt kustom, tampak seksi dari awal. sumber : CMDER.net Jika anda merasa Command Prompt bawaan dari windows sangat membosankan dan hanya mempunyai sedikit variasi perintah, kalian bisa memakai CMDER untuk menangani masalah tersebut, beberapa perintah linux juga di sisipkan di emulator ini.
